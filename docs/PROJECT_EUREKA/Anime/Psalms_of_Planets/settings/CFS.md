# 魂魄反馈系统(CFS)

[コンパクフィードバックシステム / Compac Feedback System]

<AcgImage src="/imgs/settings/CFS.jpg" title="CFS" width="230px" />

魂魄反馈系统是由塔州联邦军开发的操纵辅助系统。

使用新浪潮训练生进行测试后，最终完成。

开发过程中需要有高技术的操作能力的驾驶员。

通过直接连接到驾驶员的身体，提高 LFO 操纵性能的系统。这样一来，即使没有特殊的技术也可以操纵。最新的 Mon-Soono 机型的标配。霍兰德在剧中使用的 909、303 系统是初期型，比最新型更强。但是，那个使用给驾驶员的身体带来副作用。
